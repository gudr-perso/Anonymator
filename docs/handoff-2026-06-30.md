## Pièges connus

- **Qt QSS et `#RRGGBBAA`** : non supporté → toujours utiliser `rgba(r, g, b, alpha)` dans les styles.
- **`QLabel { background: transparent }`** : indispensable dans le QSS global, sinon les labels héritent du fond blanc de `QWidget`.
- **`sniff_delimiter`** : le fix strip la dernière ligne partielle (`text[:4096]` tronque) — ne pas revenir en arrière.
- **`Entity` est un frozen dataclass** : ne pas essayer de muter les champs, utiliser `replace()`.
- **Pillow requis** pour générer le `.ico` — vérifier qu'il est dans les dépendances (`pip show pillow`).
